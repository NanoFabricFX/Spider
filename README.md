# Spider
跨平台(Window、Linux、嵌入式设备)的物联网设备采集平台。

## 支持协议列表
1. MQTT
2. OPC UA
3. Modbus

## 运行环境
系统采用.net 5 平台开发,实现了跨平台性，可以运行在桌面系统以及嵌入设备中（例如:树莓派等）。
* 运行系统：可部署在window、Linux等操作系统中,也可以部署在Docker中。 
* 开发系统：采用图形化的配置界面、CS结构,暂定采用桌面的方式实现，运行在Window系统中。
